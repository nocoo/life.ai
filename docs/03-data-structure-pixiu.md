# 03 pixiu 记账数据结构

## 概览
这份数据来自 pixiu 记账软件导出，按年份拆分为 CSV 文件，每行代表一笔交易。

## 目录结构
```
/data
└── pixiu/
    ├── 2022.csv
    ├── 2023.csv
    ├── 2024.csv
    └── 2025.csv
```

## CSV 字段
字段采用逗号分隔，首行为表头。

- `日期`：交易日期（YYYY-MM-DD）
- `交易分类`：一级分类（例如：日常支出、日常收入）
- `交易类型`：二级分类（例如：超市、工资）
- `流入金额`：收入金额（小数）
- `流出金额`：支出金额（小数）
- `币种`：币种（例如：人民币）
- `资金账户`：资金账户名称
- `标签`：可选标签（例如：周期费用、工资）
- `备注`：可选备注

## 规则说明
- 同一行只会有 `流入金额` 或 `流出金额` 之一为非零
- `交易分类` 表示收支方向（收入/支出）
- `交易类型` 为二级分类，可用于统计与聚合
